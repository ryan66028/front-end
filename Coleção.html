Introdução a Redes de Computadores

Nota de Aula #01 - O Básico do Mundo Conectado

1. O que é uma Rede?

De forma simples, uma Rede de Computadores é um conjunto de dispositivos (computadores, celulares, servidores) conectados entre si para compartilhar recursos e trocar informações.

Analogia: Pense na rede como o sistema de correios. Para uma carta chegar, você precisa de um endereço, de um transporte e de uma linguagem que ambos entendam.

2. Componentes Principais

  • Dispositivos Finais (Hosts): Onde a informação começa ou termina (seu PC, Smart TV).
  • Dispositivos Intermediários: Quem guia os dados (Roteadores, Switches).
  • Meios de Transmissão: O caminho físico (Cabos de fibra óptica, Wi-Fi).
